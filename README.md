# Simple-Notes-App-with-React

Initial setup:
```shell
npm create vite@latest simple-notes-app --template react-ts
```
then select React >> Type script

```shell
cd simple-notes-app
npm install
```


Start the frontend
```shell
cd .\simple-notes-app\
npm run dev
```

Start the backend
```shell
cd .\backend\
node dist/server.js
```
